# Code of Conduct

This repository aims to be a respectful and usable public project.

## Expected Behavior

- Be respectful, constructive, and evidence-based.
- Focus criticism on code, behavior, and process, not on people.
- Avoid sharing private or security-sensitive information in public threads.
- Assume good intent while still being precise and honest.

## Unacceptable Behavior

- Harassment, hate speech, discrimination, or intimidation
- Personal attacks or repeated bad-faith disruption
- Publishing private information without permission
- Turning security or support channels into spam or abuse

## Reporting

For conduct problems, contact the repository maintainers through the support path in
[`SUPPORT.md`](./SUPPORT.md).

For security issues, use the private process in [`SECURITY.md`](./SECURITY.md).
